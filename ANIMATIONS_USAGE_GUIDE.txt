🎨 HOW TO USE ANIMATIONS IN YOUR HTML

═══════════════════════════════════════════════════════════════════════

1️⃣  FADE IN ANIMATIONS
───────────────────────────────────────────────────────────────────────

<div class="animate-fadeIn">Content fades in</div>
<div class="animate-fadeInUp">Content fades in from bottom</div>
<div class="animate-fadeInDown">Content fades in from top</div>
<div class="animate-fadeInLeft">Content fades in from left</div>
<div class="animate-fadeInRight">Content fades in from right</div>

───────────────────────────────────────────────────────────────────────

2️⃣  SCROLL-TRIGGERED ANIMATIONS
───────────────────────────────────────────────────────────────────────

<div class="scroll-reveal">Reveals when you scroll to it</div>
<div class="scroll-reveal-left">Slides in from left on scroll</div>
<div class="scroll-reveal-right">Slides in from right on scroll</div>

These automatically activate when user scrolls!

───────────────────────────────────────────────────────────────────────

3️⃣  HOVER EFFECTS
───────────────────────────────────────────────────────────────────────

<div class="hover-lift">Lifts up on hover</div>
<div class="hover-grow">Grows bigger on hover</div>
<div class="hover-shrink">Shrinks on hover</div>
<div class="hover-rotate">Rotates on hover</div>
<div class="hover-glow">Glows on hover</div>

───────────────────────────────────────────────────────────────────────

4️⃣  SPECIAL EFFECTS
───────────────────────────────────────────────────────────────────────

<div class="animate-bounce">Bounces continuously</div>
<div class="animate-pulse">Pulses continuously</div>
<div class="animate-shake">Shakes (great for errors)</div>
<div class="animate-wiggle">Wiggles</div>
<div class="animate-heartbeat">Heartbeat effect</div>
<div class="animate-float">Floats up and down</div>
<div class="animate-glow">Glowing effect</div>

───────────────────────────────────────────────────────────────────────

5️⃣  BUTTON EFFECTS
───────────────────────────────────────────────────────────────────────

<button class="ripple-effect">Click me!</button>
<button class="hover-lift">Lifts on hover</button>

───────────────────────────────────────────────────────────────────────

6️⃣  CARD ANIMATIONS
───────────────────────────────────────────────────────────────────────

<div class="card card-hover-effect">
    Beautiful card with hover animation
</div>

───────────────────────────────────────────────────────────────────────

7️⃣  LOADING ANIMATIONS
───────────────────────────────────────────────────────────────────────

<i class="fas fa-spinner animate-spin"></i>
<i class="fas fa-circle-notch animate-rotate"></i>

───────────────────────────────────────────────────────────────────────

8️⃣  DELAY ANIMATIONS (Stagger Effect)
───────────────────────────────────────────────────────────────────────

<div class="animate-fadeInUp delay-100">First (0.1s delay)</div>
<div class="animate-fadeInUp delay-200">Second (0.2s delay)</div>
<div class="animate-fadeInUp delay-300">Third (0.3s delay)</div>

Available delays: 100, 200, 300, 400, 500, 600, 700, 800, 900, 1000

───────────────────────────────────────────────────────────────────────

9️⃣  ANIMATION DURATION
───────────────────────────────────────────────────────────────────────

<div class="animate-fadeIn duration-fast">Fast (0.3s)</div>
<div class="animate-fadeIn duration-normal">Normal (0.6s)</div>
<div class="animate-fadeIn duration-slow">Slow (1s)</div>
<div class="animate-fadeIn duration-slower">Slower (2s)</div>

🚀 JAVASCRIPT API (Advanced Usage)

═══════════════════════════════════════════════════════════════════════

All animations auto-initialize, but you can also use JavaScript:

1️⃣  Show Toast Notification
───────────────────────────────────────────────────────────────────────
NutriLogicAnimations.toast('Success!', 'success');
NutriLogicAnimations.toast('Error!', 'error');
NutriLogicAnimations.toast('Info!', 'info');

2️⃣  Confetti Effect (Celebration!)
───────────────────────────────────────────────────────────────────────
NutriLogicAnimations.confetti();

3️⃣  Counter Animation
───────────────────────────────────────────────────────────────────────
const element = document.getElementById('counter');
NutriLogicAnimations.counter(element, 0, 100, 2000);

4️⃣  Progress Bar
───────────────────────────────────────────────────────────────────────
const bar = document.querySelector('.progress-bar');
NutriLogicAnimations.animateProgress(bar, 75, 1000);

5️⃣  Typing Effect
───────────────────────────────────────────────────────────────────────
const element = document.getElementById('text');
NutriLogicAnimations.typeWriter(element, 'Welcome to NutriLogic!', 50);

6️⃣  Shake Element (Error Feedback)
───────────────────────────────────────────────────────────────────────
const input = document.getElementById('username');
NutriLogicAnimations.shake(input);

7️⃣  Loading State
───────────────────────────────────────────────────────────────────────
const button = document.getElementById('submit');
const stopLoading = NutriLogicAnimations.showLoading(button);
// Later:
stopLoading(); // Restores button

8️⃣  Stagger Animation
───────────────────────────────────────────────────────────────────────
const items = document.querySelectorAll('.item');
NutriLogicAnimations.stagger(items, 'animate-fadeInUp', 100);
